setDialogOptions
setDialogOptions int setting, int value

Changes additional settings for BlueGUI dialogs.

Currently the only setting that can be changed through this command is the character which separates each filename when the user selects multiple files in an open file dialog.

1 - Set separator between each filename when multiple files are returned. value should be set to the ASCII character code of the desired character (which is a comma by default). You can get this value using the asc function (eg. setDialogOptions 1,asc("|") would change the character to a pipe.)

Parameters:
setting  The setting to change, possible values:
value  The value for the new setting.
