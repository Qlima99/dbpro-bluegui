setGadgetAlign
setGadgetAlign handle gadget, int alignStyle

Description:
Specifies how the gadget fits into its parent gadget's space. By default, alignment is off (ALIGN_NONE). The gadget will not move or reposition if the parent is resized. The constants file provides various codes for different styles of alignment.
  • ALIGN_NONE - No alignment
  • ALIGN_ALL - Gadget takes up all of parent's area
  • ALIGN_TOP - Gadget takes up top of parent's area
  • ALIGN_LEFT - Gadget takes up left of parent's area
  • ALIGN_RIGHT - Gadget takes up right of parent's area
  • ALIGN_BOTTOM - Gadget takes up bottom of parent's area
Return Value:
Parameters:
Remarks:
You may need to call doGadgetAlign after calling setGadgetAlign to see an immediate effect.
