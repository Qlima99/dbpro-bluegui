setGadgetColor
setGadgetColor handle gadget, color background, color text

Description:
Sets the background and text colours used by a gadget. This only works for certain types of gadget:
  • Labels
  • Panels
  • Edits
  • Listboxes
  • Comboboxes
  • Scrollbars
For more advanced customisation of gadgets, use canvas gadgets instead. To set the font style of a gadget, use createFont and applyFont.

Return Value:
Parameters:
gadget Handle of a gadget
background The background colour
text The text colour
