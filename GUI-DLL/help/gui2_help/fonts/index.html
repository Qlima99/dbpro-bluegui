Command Reference \ Font Commands
applyFont
 Applies a font to an existing gadget.
createFont
 Creates a new font.
deleteFont
 Deletes a font.
getFontName
 Retrieves the name of a font.
getFontSize
 Retrieves the point-size of a font.
isFontBold
 Returns whether or not a font is bold.
isFontItalic
 Returns whether or not a font is italic.
isFontUnderlined
 Returns whether or not a font is underlined.
