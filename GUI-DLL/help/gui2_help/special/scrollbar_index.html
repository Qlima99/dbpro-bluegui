Command Reference \ Scrollbar Commands
setScrollbarPosition
 Sets the position of the slider in a scrollbar.
setScrollbarRange
 Sets the range of possible values for the slider.
setScrollbarThumbSize
 Changes the size of the slider in a scrollbar.
getScrollbarPosition
 Retrieves the position of the slider in a scrollbar.
getScrollbarMin
 Retrieves the minimum value of the scrollbar range.
getScrollbarMax
 Retrieves the maximum value of the scrollbar range.
getScrollbarThumbSize
 Retrieves the size of a scrollbar's slider.
