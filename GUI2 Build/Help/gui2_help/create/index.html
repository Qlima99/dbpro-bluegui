Command Reference \ Create Commands
copyGadget
 Creates a copy of an existing gadget
createButton
 Creates a new button gadget.
createCanvas
 Creates a new canvas gadget.
createCheckbox
 Creates a new checkbox with a caption.
createComboBox
 Creates a new drop-down combo box.
createEdit
 Creates a new edit (or "text box") gadget.
createFrame
 Creates a new frame gadget.
createGadgetFromWindow
 Converts an existing window into a gadget.
createLabel
 Creates a new label.
createListBox
 Creates a new list box gadget.
createListView
 Creates a new list view gadget which can display icons and tables.
createPanel
 Creates a new panel gadget.
createProgressBar
 Creates a new progress bar gadget.
createRadioButton
 Creates a new radio button with a caption.
createRichEdit
 Creates a more advanced edit gadget which supports multiple fonts and colours.
createScrollbar
 Creates a new scrollbar gadget.
createStatusbar
 Creates a new statusbar.
createTabs
 Creates a new tabs gadget.
createToolbar
 Creates a new toolbar.
createTrackbar
 Creates a new trackbar or slider gadget.
createTreeView
 Creates a new treeview gadget.
createUpDown
 Creates a new UpDown spinner and attaches it to an existing gadget.
createWebBrowser
 Creates a new web browser gadget for viewing web pages.
createWindow
 Creates a new window.
