Command Reference
Create
Commands to create gadgets.
General
General commands which are applicable to all gadget types.

Special
Commands which are specific to certain types of gadget

Events
Commands used to detect user interaction with gadgets.

Fonts
Commands used to create and apply fonts to gadgets.

Menus
Commands used to create and manipulate menus and attach them to gadgets

Dialogs
Standard windows dialogs like open, save, colour, message box.

Other
Commands used to return information about the application and the computer.
