setWordWrapMode
setWordWrapMode handle gadget, bool wordWrap

Enables or disables word wrapping for a rich-edit gadget. If word-wrapping is disabled, the control can be scrolled horizontally to reveal text which extends beyond the visible area.

Word wrapping for ordinary edit gadgets can only be set at creation time by specifying the word-wrap flag (4) in the settings parameter of createEdit.

Parameters:
gadget  Handle of a rich-edit gadget
wordWrap  1 to enable word-wrapping, 0 to disable word-wrapping.
