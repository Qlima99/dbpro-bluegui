Command Reference \ Tabs Commands
addTab
 Adds a tab to a tabs gadget.
removeTab
 Removes a tab.
changeTab
 Modifies the text of a tab.
tabCount
 Gets the number of tabs in a tabs gadget.
tabCaption
 Retrieves the text of a tab item.
selectTab
 Selects a tab in a tabs gadget.
selectedTab
 Retrieves the index of the selected tab.
