desktopWindow
handle window=desktopWindow( )

Description:
Returns the window handle of the desktop. Pass this to createGadgetFromWindow to create a gadget which represents the desktop. This has many uses. For example if you wanted to get the screen resolution of the user's PC. Create a gadget representing the desktop, and call gadgetWidth and gadgetHeight on the gadget.

Return Value:
Parameters:
Remarks:
You should not, under any circumstances attempt to delete the gadget. In general, it is best only to attempt to get information about the desktop, trying to change the desktop in any way could cause problems for other applications.
