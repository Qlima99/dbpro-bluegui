enableXPStyles
enableXPStyles handle gadget, bool xpStyles

Enables or disables XP visual styles for a specific gadget. Whether or not the application uses visual styles when running under Windows XP depends on whether or not there is an appropriate manifest file in the application directory (see the BlueGUI Guide).

Parameters:
gadget  Handle of a gadget.
xpStyles  1 to enable themes for the gadget, 0 to disable visual styles for the gadget.
