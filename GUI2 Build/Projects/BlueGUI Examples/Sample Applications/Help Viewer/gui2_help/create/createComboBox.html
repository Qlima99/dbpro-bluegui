createComboBox
int handle=createComboBox(int x, int y, int width, int height, handle parent)

Description:
Creates a combobox. Comboboxes are similar to listboxes except that only one item is visible on screen at once. The others are accessed from a drop down box. Whichever item is selected is shown in the box when it is closed.

Return Value:
Handle to the new gadget.

Parameters:
x X position of new gadget
y Y position of new gadget
width Width of new gadget in pixels
height Height of new gadget in pixels
parent Handle to the parent of the new gadget, or 0 for the main window
Remarks:
The same commands are used to manage listbox items as combobox items.
